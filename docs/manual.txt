[FJT / CTNET] 二择 使用说明书

本插件使用 C++ SDK Dolores 方式调用。

在使用本插件前，您必须通读本说明书，未按照说明书操作出现的任何后果，与本插件及作者无关。

无需配置任何设置，直接启用本插件即可使用，插件将会响应【/decide】命令。

向 Bot 发送 /decide <选择项 1> <选择项 2> ... <选择项 n> 命令即可，命令与选择项之间、选择项与选择项之间用空格隔开，但文末不能出现多余空格。插件将会从所有选择项中随机选择一项进行回复。

私聊时，直接发送命令；在群聊中，需要 @ Bot。
